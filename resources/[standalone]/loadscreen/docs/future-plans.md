# Future Plans

This page outlines upcoming features and enhancements planned for future releases.

## Upcoming Features

### 1. Live Configuration Mode (Developer Mode)
A toggleable in-game UI allowing creators to edit the entire loading screen directly inside FiveM:
- Live updates without restarting the resource
- Drag-and-drop UI positioning
- Real-time color/theme previews
- Live social panel configuration
- Video/background testing tools

### 2. Improved Asset Handling
- Asset validation before loading
- Automatic fallback behavior for missing icons/images

### 3. Theme Presets
Built-in themes selectable via config:
- Minimal
- Neon
- Dark Glass
- Luxury Gold

### 4. Localization Support
Full support for multi-language translations.
